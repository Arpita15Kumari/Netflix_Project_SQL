# Netflix_Project_SQL

🚀 SQL Data Analysis Project – Netflix Dataset Analysis 📊

I recently worked on an end-to-end SQL project using a Netflix dataset to strengthen my Data Analytics and SQL problem-solving skills.

In this project, I created a structured database and solved multiple business problems using PostgreSQL queries, focusing on real-world analytical scenarios.

🔍 Key Analysis Performed:
• Counted Movies vs TV Shows
• Identified the most common ratings by content type
• Analyzed Netflix content trends over years
• Found top countries producing Netflix content
• Extracted content added in the last 5 years
• Analyzed directors, actors, genres, and documentaries
• Identified long-running TV shows and longest movies
• Categorized content using keyword-based sentiment logic
• Used advanced SQL concepts like:

✅ CTEs

✅ Window Functions (RANK)

✅ String Functions

✅ Aggregate Functions

✅ Date Functions

✅ Data Cleaning Techniques

✅ UNNEST & STRING_TO_ARRAY

💡 Key Learnings from this Project:
• Improved ability to solve business problems using SQL
• Learned how to handle messy real-world datasets
• Gained hands-on experience with analytical query writing
• Understood content trend analysis and reporting logic
• Strengthened skills in PostgreSQL data transformation techniques

This project helped me understand how SQL can be used not just for querying data, but for generating actionable insights that support business decisions.
